# Gullhella Vel – Styremøtenotater

Enkel nettside for å navigere i referater fra styremøter i Gullhella Vel.

## 🔗 Nettside
Publisert via GitHub Pages: `https://[ditt-brukernavn].github.io/[repository-navn]`

## 📁 Struktur
```
index.html   ← hele nettsiden i én fil
README.md    ← denne filen
```

## ✏️ Slik legger du til nye møter

1. Last ned `index.html` fra dette repositoryet
2. Åpne filen og finn seksjonen `<!-- M[nummer]: dato -->`
3. Erstatt plassholder-innholdet med faktiske notater
4. Oppdater sideknappen i `<nav id="sidebar">`: fjern klassen `no-data` og oppdater teksten
5. Last opp den oppdaterte `index.html` tilbake til GitHub (velg "Replace file")

Siden oppdaterer seg automatisk innen 1–2 minutter etter opplasting.
